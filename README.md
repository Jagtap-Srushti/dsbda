# DSBDA Laboratory Practicals – Semester VI

This repository contains the practical assignments and implementations for the subject **Data Science and Big Data Analytics (DSBDA)** as part of Semester VI coursework.

The repository focuses on data wrangling, data analytics, statistics, text analytics, and data visualization using Python and various data science libraries.
